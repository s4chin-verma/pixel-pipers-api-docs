PIXEL PIPERS API DOCUMENTATION

Description

There are two API endpoints for getting results: /frontend and /coordinates

Base URL

The base URL for all API requests is:

https://darkneuxmodel.westindia.azurecontainer.io

Endpoints

POST /frontend

Returns the count of objects and the image URL.

Authorization

Bearer-Token sk_nbUmMedcJAfuUjGmzhErPLOgmyGNX

Body

uploadImage : upload image file

confidence_threshold : 0.1 to 0.9 can be set but 0.6 is ideal

key : 1 for numbers or 0 for dots

font_scale: 0.5 to 1

color: choose color from below

Response

Returns a JSON object with the following properties:

count : The total number of Pipes in the image.

image_url : Result Image Url cloudinary image url

Example :

Request :

https://darkneuxmodel.westindia.azurecontainer.io/frontend

Response :

    
{
  "count": "58",
  "image_url": "https://pixelpipers.jpg"
}

Endpoints

Postman

Request Body

Headers

Confidence Threshold

The confidence_threshold indicates how confident the model is while detecting the number of pipes. The range can be set from 0.1 to 0.9, but the ideal value is typically 0.6 or 0.5.

key

In the result image, there are two options showing result numbers and dots. "1" is set for numbers and "0" is for dots.
